# Olia Brand Kit

This package turns the concentric-ring logo into a reusable brand system for product, marketing, social, and sales.

Issue: [#98](https://github.com/DoraBuilds/olia-your-daily-operations/issues/98)

## Recommended Direction

`Option 1: Orbit Signal`

Why this is the lead recommendation:

- It feels operational, memorable, and premium without losing warmth.
- It uses the existing ring mark as a true system, not just a logo.
- It can flex from app UI to hospitality signage to investor-facing materials.

## Quick Usage

- Use `olia-mark-gradient.svg` on dark surfaces, hero moments, and app splash screens.
- Use `olia-mark-dark.svg` on light backgrounds.
- Use `olia-lockup-light.svg` or `olia-lockup-dark.svg` when the wordmark is needed.
- Use `olia-app-icon.svg` for app tile, favicon source, and mobile launcher concepts.
- Use `olia-social-card.svg` as the base for LinkedIn, X, product launches, and blog OG images.
