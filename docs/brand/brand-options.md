# Brand Directions

All three directions use the current ring logo and its purple-to-blue spectrum. The choice is about personality, typography, and how boldly the system shows up.

## Option 1: Orbit Signal

Recommended.

Positioning:
Olia is the operating signal behind every shift.

Feel:
Confident, precise, modern, ambient.

Best for:
SaaS product marketing, mobile UI, investor decks, launch content.

Visual moves:

- Black or ink-led backgrounds with luminous gradient accents
- Spacious layouts with strong hierarchy
- Ring patterns used like signal maps, dashboards, and fields

Type direction:

- Display: `Space Grotesk`
- Body: `Manrope`

Color balance:

- 60% deep ink and graphite neutrals
- 25% cool whites and mist
- 15% orbit gradient accents

Tagline territory:

- The signal behind every shift.
- Operations in orbit.
- Keep every location aligned.

## Option 2: Service Halo

Positioning:
Olia brings calm structure to the chaos of hospitality operations.

Feel:
Human, elevated, editorial, quietly premium.

Best for:
Hospitality partnerships, founder brand, polished web presence, sales collateral.

Visual moves:

- Soft stone and cloud backgrounds
- Gradient used sparingly as an accent halo
- More generous editorial typography and softer spacing

Type direction:

- Display: `Fraunces`
- Body: `Manrope`

Color balance:

- 50% stone and cloud neutrals
- 30% charcoal
- 20% gradient glow

Tagline territory:

- Calm operations for busy teams.
- Hospitality standards, made repeatable.
- Shift clarity without the chaos.

## Option 3: Precision Loop

Positioning:
Olia is a disciplined system for repeatable operational excellence.

Feel:
Sharper, more technical, systems-first.

Best for:
Admin product surfaces, data-driven messaging, enterprise sales, implementation guides.

Visual moves:

- Dark navy and graphite grids
- Cleaner geometry and tighter spacing
- Ring motif used as target, progress, and status language

Type direction:

- Display: `Sora`
- Body: `IBM Plex Sans`

Color balance:

- 65% dark structure
- 20% cloud and silver
- 15% gradient signal accents

Tagline territory:

- Repeatable operations, every shift.
- Precision for the front line.
- Standards your team can actually follow.

## Recommendation

Choose `Orbit Signal`.

It is the strongest brand bridge between product and promise. It keeps the logo feeling intentional, gives Olia a recognizable signature in crowded feeds, and avoids drifting into either generic hospitality softness or generic B2B seriousness.
